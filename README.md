# MexCog-Wave-1-Factor-Structure

1: Renaming Procedures.
      This document describes which variables were pulled from https://www.mhasweb.org/DataProducts/AncillaryStudies.aspx and renamed. Reference this to understand the analytical code. 

2: Unidimensional and Hierarchical Models.
      This depicts the final models that were run to confirm the factor structure. The general cognitive functioning factor score was saved from this hierarchical model. 

3: Unidimensional Models - SVALUES.
      This depicts the SVALUES asked and implemented to save the unidimensional cognitive factor scores per domain. 

4: Correlation Analyses. 
      This code shows what was run for factor score correlations, as well as with age, education, and restricted spline analyses between scores and education. 

5: Sensitivity Analyses.
      Two sensitivity analyses were run, one incorporating household id (hhid) as a cluster [Models with hhid], and another among individuals who were administered the entire Mex-Cog cognitive protocol [Models Among Long Battery Only].
